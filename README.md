# Automated Stock & Cybersecurity News Aggregator
# Rastreador Automatizado de Noticias Financieras

A Python-based automation tool that tracks real-time breaking news for specific publicly traded companies, sending instant push notifications via a Telegram bot. Designed to monitor market-moving events efficiently.

Un proyecto de automatización en Python que rastrea noticias de última hora en tiempo real sobre empresas cotizadas en bolsa, enviando notificaciones push instantáneas a través de un bot de Telegram. Diseñado para tener información o noticias de una empresa de la cual compré acciones.
